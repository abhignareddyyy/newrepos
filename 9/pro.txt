9) API Request Format

The weather API URL format will be like:

https://api.openweathermap.org/data/2.5/weather?q=CITY_NAME&appid=YOUR_API_KEY&units=metric
Example:
https://api.openweathermap.org/data/2.5/weather?q=Hyderabad&appid=YOUR_API_KEY&units=metric
Important parts:
q=Hyderabad в†’ city name
appid=YOUR_API_KEY в†’ your key
units=metric в†’ temperature in Celsius
10) Data You Need from API Response

From the API response, extract:

name в†’ city name
main.temp в†’ temperature
main.humidity в†’ humidity
weather[0].description в†’ condition
